# Button

Button stories are based on ButtonVariant.

you can provide different props:

- loading props change text to spinner
- adding disable attribute make the button disable
- borderer props get a style border change with ButtonVariant
- shadow props add box-shadow
- you can add all button attribute , also className
- label props is the text inside the button
