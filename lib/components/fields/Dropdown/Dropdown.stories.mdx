# Dropdown

Dropdown stories are based on variants.

you can provide different props:

- placeholder: default label inside the button
- variants: change style of the component
- bs props add box-shadow
- you can add class container
- get the value selected with the onChange function
- options: is an array of datas you provide inside the list
